# Random Password Generator

**Автор:** Киселев Константин

## Описание

Приложение для генерации случайных паролей с графическим интерфейсом. Позволяет настраивать длину и состав пароля (цифры, буквы, спецсимволы), сохраняет историю сгенерированных паролей в файл JSON.

## Технологии

- Python 3.x.
- PyQt5.
- random.
- json.
- Git.

## Установка

1. Клонируйте репозиторий:
   ```
   git clone <ссылка_на_репозиторий>
   ```
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```
   python main.py
   ```

## Использование

1. Настройте длину пароля с помощью ползунка.
2. Выберите типы символов (цифры, буквы, спецсимволы).
3. Нажмите «Сгенерировать».
4. Сгенерированный пароль появится в поле и будет добавлен в историю.
5. История отображается в таблице внизу окна.

## Примеры использования

- Генерация пароля из 16 символов с цифрами и буквами.
- Генерация сложного пароля с использованием спецсимволов.
- Просмотр истории ранее сгенерированных паролей.
